# ДВИЖ — город в движении

Молодёжная соцсеть локальных активностей, челленджей и объявлений.

## Дизайн

- **Anton** + **Epilogue** — громкая молодёжная типографика
- Палитра: rose `#E11D48` + electric blue `#2563EB` + neon yellow акценты
- Sticker-UI, brutal shadows, marquee тегов
- Gen-Z energy с WCAG AA

## Запуск

```bash
npm install
npm run db:reset
npm run dev
```

**Демо:** `demo@dvizh.app` / `demo1234`
