## Фаза 0 — Фундамент и навигация

## Фаза 2 — Экран «Сегодня» (ядро)

### 2.1 Структура экрана

- [x] **M** Шапка: логотип, слоган, уровень, XP, маскот, поиск/уведомления
- [x] **M** Карточка маскота: динамические подсказки (без кринж-текстов)
- [x] **M** Карточка: XP до уровня, прогресс-бар, текущая серия
- [x] **M** Информационная подсказка дня (почему Приоритет / Сегодня ещё)
- [x] **M** Блок «Приоритет» + кнопка «Все задачи»
- [x] **M** Быстрые разделы: Календарь · Медиалист · Вишлист · Спор · Вместе
- [x] **M** Блок «Сегодня ещё» (только дела/события с датой **и** временем на сегодня)
- [x] **W** Аналог экрана «Сегодня» в веб-профиле/дневнике

### 2.2 Карточка задачи (правило v23+)

- [x] **W+M** Чекбокс слева = выполнение (отдельная зона клика)
- [x] **W+M** Буферная зона между чекбоксом и текстом (ничего не делает)
- [x] **M** Тело карточки = редактирование
- [x] **W+M** Показывать: номер в приоритете, название, хештег, видимость, 🔥 N, XP справа один раз
- [x] **W+M** Не показывать: пруф, время, дату, напоминание, дубль XP, длинную серию текстом
- [x] **W+M** После выполнения: галочка, мягкое зачёркивание, XP, лёгкий эффект/конфетти
- [x] **M** `CelebrationLayer` не блокирует тапы

### 2.3 Фото-пруф и ачивки после выполнения

- [x] **M** Окно «Добавить фото-пруф?» только после выполнения (если включено в задаче/настройках)
- [x] **W+M** Фото не публикуется автоматически; выбор видимости после загрузки
- [x] **W+M** Баннер «Ачивка разблокирована» снизу + лёгкое конфетти
- [x] **B** API загрузки фото-пруфа к задаче

## Фаза 3 — Задачи: создание, редактирование, список

- [x] **W+M** Поля: название, заметка, хештег (#спорт, #быт, …)

### 3.2 Когда выполнить

- [x] **W+M** Отдельный тумблер «Добавить время» (по умолчанию выкл.)
- [x] **W+M** Логика попадания в календарь / «Сегодня ещё» по правилам v24
- [x] **B** Поля `scheduledAt`, `hasTime`, timezone rollover

### 3.3 Повтор, поведение, видимость

- [x] **W+M** Повтор: день / неделя / месяц / год / свой
- [x] **W+M** Тумблеры: Приоритет · Напомнить пушем · Спросить про фото-пруф
- [~] **M** Локальные пуш-напоминания (если в scope мобилки)
- [x] **B** `priority`, `askProof` — синхронизация с мобилкой

### 3.4 Чек-лист

- [x] **W+M** Типы: Покупки · В дорогу · Гости · Подарки · Свой
- [x] **W+M** CRUD пунктов чек-листа
- [x] **W+M** Медиа-категории только в Медиалисте, не в чек-листе задачи

### 3.5 Все задачи

- [x] **M** Экран «Все задачи» из блока Приоритет
